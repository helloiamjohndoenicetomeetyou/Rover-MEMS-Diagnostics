## System Requirements
* Android Version: Android 8.0 (Oreo) or Later (API Level 26 or Higher)
* Connectivity: USB OTG Support

## Supported ECU Version
* MEMS 1.6 (Commonly found in Classic Mini SPI)

## Features
* Live Data
* Fault Codes
* Tuning
